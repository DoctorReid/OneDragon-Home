# 一条龙主页

[VuePress](https://vuepress.vuejs.org/zh/)

## 开发环境

[nodejs v20.17.0](https://nodejs.org/dist/v20.17.0/node-v20.17.0-win-x64.zip)

淘宝源
```npm
npm config set registry https://registry.npmmirror.com
```

初始化
```npm
pnpm install
```

本地运行
```npm
pnpm docs:dev
```

更新版本
```npm
pnpm docs:update-package
```

打包
```npm
pnpm docs:build
```
